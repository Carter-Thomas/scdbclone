COLL

3x3

Description

COLL or Corners of Last Layer, is a last layer algorithm set for the case where all edges are oriented. This set both orient and permutes the corners of the last layer.

Applying COLL algorithm, will result in either an edge only PLL which are considered more favorable (Ua, Ub, H and Z) or a skip in 1/12 of the cases.
