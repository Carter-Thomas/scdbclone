ZBLL

3x3
Description
ZBLL or Zborowski-Bruchem Last Layer is an algorithm set that aims to solve the entire last layer in a single algorithm, in the case where the edges are already oriented. Many choose to skip learning the Sune and AntiSune subsets since their OLL cases are already short and easy to execute.
